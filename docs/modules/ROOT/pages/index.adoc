== Installation

If you want to use this extension, you need to add the `com.crossvale:entity-extension`
extension first to your build file.

For instance, with Maven, add the following dependency to your POM file:

[source,xml,subs=attributes+]
----
<dependency>
    <groupId>com.crossvale</groupId>
    <artifactId>entity-extension</artifactId>
    <version>{project-version}</version>
</dependency>
----
